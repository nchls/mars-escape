# Mars Escape
This is a React JS resource management game about Mars rovers made for React Riot 2019.

## Playing
Visit https://nchls.github.io/mars-escape/

## Running locally
* `npm install`
* `npm start`
* Open http://127.0.0.1:8080

## Deploying
* Checkout `master`
* `npm run build`
* Commit the changes and push
